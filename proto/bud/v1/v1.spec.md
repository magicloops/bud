# v1

Version 1 Bud daemon-service protobuf schema.

## Files

### `bud.proto`

Defines `BudEnvelope`, traffic/transport enums, typed errors, operation and stream lifecycle enums, reconnect/reconciliation payloads, current terminal/control payloads, and a `LegacyJsonPayload` compatibility payload retained for downgrade/fixture support.

Known current payload messages include a transitional `frame_json` field so WebSocket-capable peers can dispatch by typed oneof payload tag before the service and daemon switch fully to generated field-level protobuf structs.
