# assets

Static assets bundled with the application.

## Purpose

Contains static image assets that are imported and bundled by Vite during build.

## Files

### `react.svg`

React logo SVG. Default Vite template asset.

**Usage**: Not currently used in production UI.

## Notes

Assets in this folder are:
- Processed by Vite's asset pipeline
- Given content-hashed filenames in production builds
- Can be imported as modules: `import reactLogo from '@/assets/react.svg'`
