# v1

Version 1 protobuf schema for the gRPC interop spike.

## Purpose

This package defines the small control and attach services used to validate Rust tonic interoperability with the Node Connect and grpc-js gateway candidates.

## Files

- [interop.proto](./interop.proto) - `BudControlInteropService.Connect`, `BudAttachInteropService.Attach`, and the minimal control/data frame messages.

## Dependencies

- [../../../../buf.yaml](../../../../buf.yaml) - Buf module and lint rules.
- [../../../../buf.gen.yaml](../../../../buf.gen.yaml) - TypeScript generation template.
- [../../../../daemon/build.rs](../../../../daemon/build.rs) - Rust generation entrypoint.
